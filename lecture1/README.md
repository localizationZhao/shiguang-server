# lecture1

## 目录结构

```
lecture1/
├── lecture1_课后作业/    # 课后作业：简·爱书籍展示
└── lecture1_课堂作业/    # 课堂作业：小程序现状调研
```

## 子文件夹概述

### lecture1_课后作业

简·爱书籍展示小程序。展示简·奥斯汀与其著作《简爱》的相关信息，包含作者生平、著作背景、作品价值等内容，使用微信小程序原生开发实现页面布局与内容展示。

### lecture1_课堂作业

小程序现状调研报告。调研市场上各种小程序的现状和共同之处，包括微信、支付宝、百度、抖音等平台小程序，分析其无需安装、轻量化、社交传播、支付闭环等共同特征。
