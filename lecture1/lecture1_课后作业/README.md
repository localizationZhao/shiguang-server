# lecture1_课后作业项目：《简·爱》图书详细介绍页

#### 项目路径
`jane_eyre_book/`

#### 项目简介
基于微信小程序原生框架开发的《简·爱》图书详情展示页面，纯静态前端项目，无后端接口、无数据库交互，所有数据均本地静态写入。

#### 技术栈
- 微信小程序原生框架（WXML + WXSS + JS + JSON）
- rpx自适应单位
- Flex布局

#### 页面包含内容
1. **封面模块**：书籍封面图片 + 标题 + 9.8高分标签
2. **基础信息**：作者、出版社、出版年份、类型、页数、适配人群、标签
3. **书籍简介**：完整内容简介
4. **作者简介**：夏洛蒂·勃朗特详细介绍
5. **章节目录**：8章精简版梗概

#### 项目结构
```
jane_eyre_book/
├── app.js           # 全局应用入口
├── app.json         # 全局配置文件
├── app.wxss         # 全局样式文件
├── sitemap.json     # 站点地图配置
├── images/          # 图片资源
│   └── image.png    # 书籍封面
└── pages/
    └── index/
        ├── index.js    # 页面逻辑
        ├── index.json  # 页面配置
        ├── index.wxml  # 页面结构
        └── index.wxss  # 页面样式
```

#### 使用方法
1. 打开微信开发者工具
2. 导入项目（选择 `jane_eyre_book` 文件夹）
3. 点击编译即可运行
