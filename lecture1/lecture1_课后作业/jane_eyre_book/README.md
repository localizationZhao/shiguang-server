# 《简·爱》图书详细介绍页 - 微信小程序项目

## 项目简介
本项目是一个基于微信小程序原生框架开发的《简·爱》图书详情展示页面，纯静态前端项目，无后端接口、无数据库交互，所有数据均本地静态写入。

## 项目特点
- **设计风格**：简约文艺阅读风，莫兰迪浅米色+深灰文字配色，低饱和度、治愈系
- **页面结构**：六大核心模块（封面、基础信息、内容简介、作者简介、章节目录）
- **适配兼容**：rpx自适应单位，完美适配iOS/安卓全尺寸手机
- **零依赖**：无需登录授权、无需网络请求、无需跳转，开箱即用

## 项目结构
```
jane_eyre_book/
├── app.js           # 全局应用入口
├── app.json         # 全局配置文件
├── app.wxss         # 全局样式文件
├── sitemap.json     # 站点地图配置
└── pages/
    └── index/
        ├── index.js    # 页面逻辑
        ├── index.json  # 页面配置
        ├── index.wxml  # 页面结构
        └── index.wxss  # 页面样式
```

## 使用方法
1. 打开微信开发者工具
2. 导入项目（选择 `jane_eyre_book` 文件夹）
3. 点击编译即可运行

## 页面展示内容
- 图书封面与标题（含评分标签）
- 图书基础信息（作者、出版社、出版年份、类型、页数等）
- 完整书籍内容简介
- 作者详细介绍
- 精简章节目录梗概（共8章）

## 技术栈
- 微信小程序原生框架（WXML + WXSS + JS + JSON）
- rpx自适应单位
- Flex布局
