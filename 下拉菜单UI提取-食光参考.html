🎨 配色体系

变量值说明
--birb-border-size2px边框粗细
--birb-highlight#ffa3cb高亮色（标题栏 + 边框）
--birb-background-color#ffecda底色（窗口 + 菜单项）

换掉高亮色 = 换一套主题。比如食光小程序可以用 #ff4997。

🏗️ 结构

birb-window                   ← 弹出窗口容器（像素双层边框）
├── birb-window-header        ← 彩色标题栏
├── birb-window-content       ← 奶油色内容区
│   ├── birb-menu-item        ← 普通菜单项
│   ├── birb-menu-item.has-submenu  ← 带子菜单的项
│   │   └── birb-window.submenu     ← 子菜单（悬浮在右侧）
│   └── birb-window-separator ← 分隔线
└── (birb-menu-item-spinner)  ← 数字调节器（+/-按钮）

💡 核心技巧

1. 像素风双层边框（和便签一样的技法）

box-shadow:
  2px 0 var(--hl), -2px 0 var(--hl),   /* 内层彩色 */
  0 -2px var(--hl), 0 2px var(--hl),
  4px 0 var(--hl), -4px 0 var(--hl),   /* 外层彩色 */
  0 -4px var(--hl), 0 4px var(--hl),
  0 0 0 2px var(--hl),                  /* 缝隙填充 */
  0 0 0 4px #fff,                       /* 白色间隙 ← 像素感关键 */
  ...

2. 悬浮高亮反转

.birb-menu-item { opacity: 0.7; color: black; }
.birb-menu-item:hover {
  opacity: 1;
  background: var(--hl);  /* 高亮色铺底 */
  color: white;           /* 文字变白 */
}

3. 弹出动画

@keyframes pop-in {
  0%   { transform: scale(0.1); opacity: 1; }
  100% { transform: scale(1);   opacity: 1; }
}

4. 子菜单定位

.has-submenu { position: relative; }
.submenu {
  position: absolute;
  left: calc(100% + 4px);  /* 紧贴父菜单右侧 */
  top: 0;
  display: none;
}
.has-submenu:hover .submenu { display: block; }

pop-in 动画小程序支持 @keyframes
:hover 伪类小程序用 hover-class 代替
子菜单 CSS hover小程序用 wx:if + bindtap 控制显示
Monocraft 像素字体转 base64 内嵌到 WXSS（见下方）
