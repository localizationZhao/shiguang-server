# 移动开发作业2026

本仓库包含 **移动开发** 课程的全部作业，涵盖微信小程序原生开发、数据绑定、事件处理、视图容器、后端API、多媒体API、支付功能等核心知识点。

## 目录结构

```
移动开发作业2026/
├── lecture1/              # 第1讲：小程序基础与调研
├── lecture2/              # 第2讲：JSON数据格式与增强版小程序
├── lecture3/              # 第3讲：视图容器与新闻小程序
├── lecture4/              # 第4讲：后端API与视频页开发
├── lecture5/              # 第5讲：多媒体API与位置服务
├── lecture6/              # 第6讲：微信支付与WebSocket
└── 个人项目/               # ★ 个人独立开发项目
    └── 食光/               # 像素风美食社交小程序
```

## 技术栈

- **前端**：微信小程序原生开发（WXML、WXSS、JavaScript）
- **后端**：Python Flask
- **数据库**：SQLite
- **支付**：微信支付API

## 运行方式

1. 使用微信开发者工具打开各作业文件夹
2. 对于 lecture4 课后作业，需启动 Flask 后端服务
3. 在微信开发者工具中预览或真机调试

## 提交说明

本仓库为课程作业提交仓库。个人独立开发项目（食光）见 `个人项目/` 目录。
