# lecture5

## 目录结构

```
lecture5/
├── lecture5_课后作业/    # 课后作业：录音播放 + 滚动多媒体
└── lecture5_课堂作业/    # 课堂作业：位置API + 表单组件
```

## 子文件夹概述

### lecture5_课后作业

课后作业项目。包含两个小程序项目：录音与播放功能开发（录音API、文件管理API、音频控制API）、左右滚动多媒体界面开发（横向滚动承载四类媒体元素，动态修改数据源）。

### lecture5_课堂作业

课堂作业项目。包含两个小程序项目：位置API演示（地图标记、路线绘制、圆形区域）、表单组件综合练习（map、checkbox、radio、switch、canvas、slider、picker、form等组件）。
