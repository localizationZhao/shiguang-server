## 子文件夹概述

### lect5_课后一_API     课后作业（一）

录音与播放功能开发。实现录音API、文件管理API、音频控制API三大核心功能，支持按住录音、文件保存、列表展示、播放控制等完整流程。

### lec5_课后二_exercise_scroll_media课后作业（二）

左右滚动多媒体界面开发。使用 scroll-view 横向滚动承载文字、图片、音频、视频四类元素，通过下方组件绑定事件动态修改数据源，实时更新滚动界面显示效果。
