# lecture5_课堂作业

## 目录结构

```
lecture5_课堂作业/
├── exercise_form/        # 课堂作业：表单组件综合练习
└── exercise_location/    # 课堂作业：位置API演示
```

## 子文件夹概述

### exercise_form

表单组件综合练习。覆盖表单组件（二）和表单组件（三）知识点，包含 map、checkbox、radio、switch、canvas、slider、picker、picker-view、form 等组件的完整实现与交互演示。

### exercise_location

位置API演示。使用 wx.getLocation、wx.chooseLocation、wx.openLocation 等位置相关API，实现地图标记、路线绘制、圆形区域等功能，需声明位置权限。
