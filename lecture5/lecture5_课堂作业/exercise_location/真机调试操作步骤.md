# 课堂作业：位置API真机调试操作步骤

## 一、微信开发者工具端操作

### 1. 打开项目
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择项目目录：`E:\移动开发刘强\课件\lecture5\lecture5_课堂作业\exercise_location`
4. AppID 填写：使用测试号（可以留空，使用微信测试号）
5. 点击「导入」

### 2. 开启调试模式
1. 点击顶部菜单栏「详情」按钮
2. 在「本地设置」中勾选：
   - ☑ 不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书
3. 确认「调试基础库」版本为 3.16.1 或更高

### 3. 生成预览二维码
1. 点击顶部工具栏「预览」按钮
2. 在弹出的对话框中，勾选「开启调试模式」
3. 点击「确定」生成二维码

## 二、手机端操作

### 1. 扫码打开小程序
1. 打开手机微信
2. 进入「发现」→「扫一扫」
3. 扫描开发者工具生成的二维码
4. 微信会提示「是否打开小程序」，点击「允许」

### 2. 授权位置权限
1. 首次打开小程序时，会弹出位置权限申请弹窗
2. 点击「允许」授予位置权限
3. 如果误点了「拒绝」，可以通过模块四的「打开设置页」重新开启

### 3. 验证四个功能模块

#### 模块一：获取当前位置
1. 点击「获取当前位置」按钮
2. 等待定位完成（可能需要几秒）
3. 验证页面显示：纬度、经度、速度、位置精度、海拔高度等数据
4. 重复点击可刷新位置数据

#### 模块二：选择地图位置
1. 点击「选择地图位置」按钮
2. 验证弹出微信内置地图选点界面
3. 在地图上搜索或拖动选择一个位置
4. 点击「确定」返回
5. 验证页面显示：位置名称、详细地址、纬度、经度

#### 模块三：打开地图位置
1. 先通过模块一或模块二获取一个位置
2. 点击「打开当前位置」或「打开选中位置」按钮
3. 验证跳转到原生地图页面，显示目标位置详情
4. 测试缩放、导航等功能

#### 模块四：位置授权管理
1. 点击「检测授权状态」按钮
2. 验证显示当前授权状态（已授权/已拒绝/未授权）
3. 如果状态为「已拒绝」，点击「打开设置页」按钮
4. 在设置中手动开启位置权限

### 4. 打开调试面板 vConsole
1. 点击小程序右上角「...」菜单
2. 点击「调试」选项
3. 底部会弹出 vConsole 调试面板
4. 切换到「Console」标签页
5. 验证无报错信息，接口调用成功

## 四、常见问题排查

### 问题1：获取位置失败，提示无权限
- 解决：确保在 app.json 中配置了 permission 和 requiredPrivateInfos
- 在手机设置中手动开启小程序位置权限

### 问题2：真机无法定位
- 解决：确保手机已开启定位服务（GPS/网络定位）
- 确保网络连接正常

### 问题3：选择位置界面不弹出
- 解决：确认授权状态为「已授权」
- 在开发者工具详情中勾选调试模式

### 问题4：打开地图失败
- 解决：确保传入的经纬度为数字类型
- 确认经纬度在有效范围内

## 五、代码结构说明

```
exercise_location/
├── app.js              # 小程序入口
├── app.json            # 全局配置（含位置权限声明）
├── app.wxss            # 全局样式
├── project.config.json # 项目配置
├── sitemap.json        # 站点地图
└── pages/
    └── location/
        ├── location.wxml    # 页面结构（四大功能模块）
        ├── location.js      # 逻辑层（位置API调用）
        ├── location.wxss    # 页面样式
        └── location.json    # 页面配置
```

### 核心API说明

| API | 功能 | 关键参数 |
|-----|------|---------|
| wx.getLocation | 获取当前位置 | type: 'gcj02', altitude: true |
| wx.chooseLocation | 选择地图位置 | 无 |
| wx.openLocation | 打开地图位置 | latitude, longitude, name, address |
| wx.authorize | 申请权限 | scope: 'scope.userLocation' |
| wx.openSetting | 打开设置页 | 无 |
| wx.getSetting | 获取授权状态 | 无 |
