💡 CSS 核心技巧

1. 右上角像素 x 关闭按钮

就是标题栏里一个小小的 x。Monocraft 像素字体让它自带锯齿感，不需要图片：

.sticky-close {
  position: absolute;
  top: 1px;
  right: 0;
  padding: 0 5px;
  color: var(--bg);           /* 和标题栏同色 → 变成反白效果 */
  font-family: Monocraft;     /* 像素字体 → x 带锯齿 */
  font-size: 16px;
  cursor: pointer;
  user-select: none;
}
.sticky-close:hover {
  transform: scale(1.1);      /* 悬浮微微放大 */
}

原理：x 就是个普通文字，放在彩色标题栏里，颜色设成底色(=反白)。像素字体让它看起来像画上去的。小程序里改成 font-family: Monocraft, monospace 同样效果。

2. 像素风双层边框（box-shadow 技法）

用多重 box-shadow 实现像素感粗边框 + 白色间隙，不是普通的 border：

/* --border = 2px, --hl = 标题栏颜色 */
box-shadow:
  2px 0 var(--hl), -2px 0 var(--hl),  /* 左右彩色内层 */
  0 -2px var(--hl), 0 2px var(--hl),  /* 上下彩色内层 */
  4px 0 var(--hl), -4px 0 var(--hl),  /* 左右彩色外层 */
  0 -4px var(--hl), 0 4px var(--hl),  /* 上下彩色外层 */
  0 0 0 2px var(--hl),                 /* 缝隙填充 */
  0 0 0 4px #fff,                      /* 白色间隙 */
  4px 0 0 2px #fff, -4px 0 0 2px #fff, /* 左右白色…… */
  …

3. CSS变量换色

5种配色不用写5份CSS，只改两个变量：

.n1 { --hl: #ff8baa; --bg: #ffecda; }  /* 粉 */
.n2 { --hl: #79bcff; --bg: #ffecda; }  /* 蓝 */
.n3 { --hl: #ffb37c; --bg: #ffecda; }  /* 橘 */

.n5 { --hl: #6de192; --bg: #ffecda; }  /* 绿 */

3. 悬浮效果

.sticky-note:hover {
  transform: scale(1.03);                          /* 微微放大 */
  filter: drop-shadow(0 8px 5px rgba(0,0,0,.15)); /* 投影浮起 */
}

🪟 附：日志/评论区的毛玻璃透明是怎么做到的

Pocket Bird 商店页面的 Devlog 和 Comments 区域是半透明毛玻璃效果。核心就一行：

.footer, .game_devlog, .game_comments_widget, .uploads {
  background: rgba(187, 210, 255, 0.15);     /* 淡蓝 · 仅15%不透明 */
  backdrop-filter: blur(10px);               /* 背景模糊 10px */
  -webkit-backdrop-filter: blur(10px);       /* Safari/小程序兼容 */
  border-radius: 25px;
}

原理

步骤发生了什么
1页面背景是渐变天空（蓝色→紫色）
2backdrop-filter: blur(10px) 把后面像素模糊
3rgba(187,210,255, 0.15) 盖一层薄薄淡蓝
4你看到 → 半透明毛玻璃，能看到背后图片但看不清

你的食光小程序已经在用了

app.wxss 里的 .glass 就是完全一样的原理：

/* 食光 — 你已经有的代码 */
.glass {
  background: rgba(255, 255, 255, 0.25);     /* 白色 · 25%不透明 */
  backdrop-filter: blur(15px);
  -webkit-backdrop-filter: blur(15px);
  border-radius: 25px;
}

Pocket Bird 用淡蓝（rgba(187,210,255,0.15)），食光用白色（rgba(255,255,255,0.25)）。换颜色就能变风格。

☁️ 云朵背景是怎么做到的

页面背景不是纯色，而是一张像素风天空插图，通过 base64 直接写在 CSS 里：

html {
  /* 渐变色打底 */
  background: linear-gradient(180deg, #66b0ff, #bfbbff);
  /* 云朵图片盖在上面 */
  background-image: url("data:image/png;base64,iVBORw0KGgoAAAA...");
  background-attachment: fixed;  /* 滚动时背景不动 */
  background-size: cover;
  background-position: bottom;
  image-rendering: pixelated;    /* 像素风不模糊 */
}

三层叠加

层是什么CSS
1. 底色天蓝→淡紫渐变linear-gradient(180deg, #66b0ff, #bfbbff)
2. 云朵base64像素画background-image: url(data:image/png;base64,...)
3. 固定滚动不跟走background-attachment: fixed

云朵就是一张PNG图片转成了字符串。小程序同样支持 background-image + base64。

🪄 "所有部件都悬浮"是怎么做到的

不是真的悬浮，是毛玻璃把前景和背景分离开了。大脑看到模糊的背景 + 清晰的前景 → 自动判断"这个东西浮在上面"。

三个因素叠加

因素代码效果
毛玻璃 backdrop-filter: blur(10px) 背景模糊 → 前景"脱离"背景
半透明底色 background: rgba(255,255,255,0.25) 能看到背后 → 不是死板的白块
投影 filter: drop-shadow(0 8px 5px rgba(0,0,0,0.15)) 下方有阴影 → 确认"浮起来了"

悬浮感的本质

┌─────────────────────────┐
│ ☁️ 云朵背景（模糊的）    │  ← 被 blur 模糊了
│  ┌─────────────────┐    │
│  │ 便签卡片（清晰的）│    │  ← 不受 blur 影响
│  │ 还有投影 ↓      │    │
│  └─────────────────┘    │
│         ░░░░░░ 阴影      │  ← drop-shadow
└─────────────────────────┘

你的食光小程序已经具备全部三个条件了——.glass 类就是毛玻璃。
