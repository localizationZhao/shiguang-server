💡 原理：三件套

部件代码作用
固定框 height: 260px; overflow-y: hidden 限定可视区域，超出部分藏起来
滚动动画 animation: scroll 8s ease-in-out infinite 内容上下缓慢移动，无限循环
渐变遮罩 linear-gradient(底色 → 透明) 顶部文字淡入，底部文字淡出，不突兀

🎯 核心CSS（完整可复制）

/* 1. 外层容器 — 定高 + 隐藏溢出 */
.scroll-container {
  height: 260px;        /* 可视高度 */
  overflow-y: hidden;   /* 超出就藏 */
  position: relative;
}

/* 2. 渐变遮罩 — 上下各一个 */
.scroll-fade-top {
  height: 40px;
  margin-bottom: -40px;  /* 压进内容区 */
  background: linear-gradient(180deg, #ffecda, transparent);
  z-index: 2;
}
.scroll-fade-bottom {
  height: 40px;
  margin-top: -40px;
  background: linear-gradient(0deg, #ffecda, transparent);
  z-index: 2;
}

/* 3. 动画 — 慢慢上下移动 */
.scroll-list {
  animation: scroll 8s ease-in-out infinite;
}
@keyframes scroll {
  0%   { transform: translateY(0); }
  50%  { transform: translateY(-60%); }  /* 上移 */
  100% { transform: translateY(0); }     /* 回落 */
}

原版小程序改法

内容（鸟名列表）换成菜谱名、分类名

用途：首页"今日推荐"轮播、餐厅"热门菜品"滚动、个人中心"我的菜谱"列表。把鸟名换成菜名就行。
