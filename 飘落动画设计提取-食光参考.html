💡 原理：三个动画同时跑

每片羽毛同时做三个动作，组合起来就是"飘飘荡荡往下掉"：

动画做什么时长
floatX左右飘荡（translateX 来回）15秒
floatY上下起伏（margin-top 波动）15秒
fall从顶掉到底（top: -80 → 420px）20秒

三个动画同时作用于一个元素，时间不同步 → 每片羽毛轨迹都不一样。

🎯 核心CSS

/* 每片羽毛：固定定位 + 三个动画叠加 */
.feather {
  position: fixed;       /* 或 absolute */
  font-size: 32px;       /* emoji 大小 */
  z-index: -1;           /* 在内容背后 */
  opacity: 0.6;          /* 半透明 */
  animation:
    floatX 15s ease-in-out infinite,   /* 左右荡 */
    floatY 15s ease-in-out infinite,   /* 上下颠 */
    fall   20s linear     infinite;    /* 往下掉 */
}

/* 每片延迟不同 → 错开不重叠 */
.feather:nth-child(1) { animation-delay:   0s,   0s,   0s; }
.feather:nth-child(2) { animation-delay:  -4s,  -4s,  -4s; }
.feather:nth-child(3) { animation-delay:  -8s,  -8s,  -8s; }
.feather:nth-child(4) { animation-delay: -12s, -12s, -12s; }
.feather:nth-child(5) { animation-delay: -16s, -16s, -16s; }

@keyframes floatX {
  0%,100% { transform: translateX(0); }
  50%     { transform: translateX(100px); }
}
@keyframes floatY {
  0%,50%,100% { margin-top: 0; }
  25%,75%     { margin-top: 60px; }
}
@keyframes fall {
  0%   { top: -80px; }
  100% { top: 420px; }    /* 比如容器高度 */
}

🍕 改成餐厅风格：飘食物和餐具

把 emoji 换掉就行，CSS 完全不动：

/* 原版 */
<div class="feather">🪶</div>

/* 食光版 */
<div class="feather">🍕</div>   ← 披萨
<div class="feather">🥢</div>   ← 筷子
<div class="feather">🍜</div>   ← 拉面
<div class="feather">🍳</div>   ← 煎蛋
<div class="feather">🥟</div>   ← 饺子
<div class="feather">🍷</div>   ← 酒杯
<div class="feather">🔪</div>   ← 菜刀

🔧 小程序版本（WXML + WXSS）

WXML

<view class="floating-bg">
  <view class="float-item" style="animation-delay:0s">🍕</view>
  <view class="float-item" style="animation-delay:-3s">🥢</view>
  <view class="float-item" style="animation-delay:-6s">🍜</view>
  <view class="float-item" style="animation-delay:-9s">🍳</view>
  <view class="float-item" style="animation-delay:-12s">🥟</view>
</view>

WXSS

.float-item {
  position: fixed;
  font-size: 40rpx;
  z-index: -1;
  opacity: 0.5;
  animation: floatX 15s ease-in-out infinite,
             floatY 15s ease-in-out infinite,
             fall   20s linear infinite;
}
/* @keyframes 同上，单位改成 rpx */

小程序完全支持 @keyframes。唯一区别：用 rpx 代替 px。
