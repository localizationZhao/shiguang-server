## 子文件夹概述

### lecture6_课后作业

课后作业：微信支付功能实现。以精品书籍购买为业务场景，完整实现支付参数构造、MD5签名计算、wx.requestPayment API调用流程，包含支付成功/失败的完整回调处理与结果展示。

### lecture6_课堂作业

课堂作业：WebSocket协议调研。包含初步调研结果和详细调研报告两部分，覆盖协议概述、与HTTP对比、工作原理、核心特性、典型应用场景、微信小程序API应用等知识点。
