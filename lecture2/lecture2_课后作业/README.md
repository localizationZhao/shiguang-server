# -Lecture 2 课后作业

## 项目概述

**项目路径**：`lecture2_jane_eyre_enhanced/`

**项目简介**：基于微信小程序原生框架开发的《简·爱》图书详情页，展示 WXML 数据绑定、列表渲染、条件渲染、模板等核心语法。

---

## 核心功能

| 模块 | 功能 | 技术点 |
|------|------|--------|
| 图书详情 | 封面、基础信息、标签、章节目录 | 数据绑定、`wx:for` |
| 简介展开/收起 | 书籍简介、作者简介 | `wx:if` + 事件交互 |
| 数据绑定示例 | Mustache 语法 12 种用法 | 属性绑定、三元运算、对象扩展等 |
| 列表渲染示例 | `wx:for` 7 种用法 | `wx:key`、嵌套列表、自定义变量名 |
| 条件渲染示例 | `wx:if` 4 种用法 | `wx:elif/else`、`block wx:if`、`hidden` 对比 |
| 模板示例 | `template` 定义与动态切换 | 按性别切换人物模板 |

---

## 原创性说明

所有示例代码围绕《简·爱》编写，未照搬官方文档：

| 官方示例 | 本项目实现 |
|---------|-----------|
| 九九乘法表 | 人物关系矩阵 |
| 数字列表 | 《简·爱》章节列表、经典名著列表 |
| 奇偶模板 | 男/女性角色模板 |

---

## 运行方法

1. 打开微信开发者工具
2. 导入 `lecture2_jane_eyre_enhanced` 文件夹
3. 编译运行
