# 《简·爱》图书详情页 - 微信小程序增强版项目

## 项目简介
本项目是基于微信小程序原生框架开发的《简·爱》图书详情展示页面，为 Lecture 2 课后作业的增强版本。项目围绕《简·爱》这本经典名著，全面展示微信小程序 WXML 语法的核心功能，包括数据绑定、列表渲染、条件渲染、模板和引用等。所有示例代码均围绕《简·爱》主题原创编写，**严禁照搬官方文档示例**。

## 技术栈
- **框架**：微信小程序原生框架（WXML + WXSS + JS + JSON）
- **单位**：rpx 自适应单位
- **布局**：Flex 弹性布局为主
- **数据驱动**：所有页面数据集中在 `index.js` 的 `data` 对象中

## 页面功能模块（共 11 个模块）

### 模块一：封面与标题区
- 封面图片（`cover` 数据绑定）
- 书名标题（`title` 数据绑定）
- **条件渲染**：根据评分 `score >= 9.5` 显示"高分推荐"或"值得一读"标签

### 模块二：基础信息列表（列表渲染）
- 使用 `<block wx:for="{{infoList}}" wx:key="label">` 渲染
- 展示作者、出版社、出版年份、ISBN、页数等信息
- 每条信息包含 `label`（标签名）和 `value`（值）

### 模块三：标签区域（列表渲染）
- 使用 `<block wx:for="{{tags}}" wx:key="*this">` 渲染
- 标签气泡样式，展示"经典名著"、"女性成长"、"英国文学"等标签

### 模块四：书籍简介（条件渲染：展开/收起）
- 通过 `showFullSummary` 布尔值控制显示完整内容或截断前 50 字
- 按钮文字动态切换：`{{showFullSummary ? '收起' : '展开全部'}}`
- 按钮点击阴影收缩效果（`hover-class` + CSS `transform: scale`）

### 模块五：作者简介（条件渲染：展开/收起）
- 通过 `showFullAuthorBio` 布尔值控制显示完整内容或截断前 50 字
- 按钮文字动态切换：`{{showFullAuthorBio ? '收起' : '展开全部'}}`
- 按钮点击阴影收缩效果

### 模块六：章节目录（列表渲染 + wx:key）
- 使用 `<block wx:for="{{chapters}}" wx:key="id">` 渲染
- 展示 8 章目录，每章包含章节名和简介
- 使用 `wx:key="id"` 提升列表渲染性能

### 模块七：电子版提示（条件渲染）
- 通过 `hasEbook` 布尔值判断显示"本书支持微信读书"或"暂无电子版"

## 核心功能实现

### 列表渲染（wx:for）使用位置
1. **基础信息列表**：`infoList` 数组通过 `<block wx:for="{{infoList}}">` 渲染
2. **标签区域**：`tags` 数组通过 `<block wx:for="{{tags}}">` 循环渲染标签气泡
3. **章节目录**：`chapters` 数组通过 `<block wx:for="{{chapters}}" wx:key="id">` 渲染
4. **默认列表渲染**：`defaultArray` 展示《简·爱》章节列表
5. **自定义变量名**：`customArray` 使用 `wx:for-index="idx"` `wx:for-item="book"`
6. **嵌套列表渲染**：`characterMatrix` 人物关系矩阵（双层 `wx:for`）
7. **block wx:for**：`bookInfoBlocks` 书籍信息块
8. **wx:key（property）**：`objectArray` 使用 `wx:key="unique"`
9. **wx:key（*this）**：`numberArray` 使用 `wx:key="*this"`

### 条件渲染（wx:if/elif/else）使用位置
1. **评分标签**：根据 `score >= 9.5` 判断显示"高分推荐"或"值得一读"
2. **书籍简介展开/收起**：通过 `showFullSummary` 布尔值控制
3. **作者简介展开/收起**：通过 `showFullAuthorBio` 布尔值控制
4. **电子版提示**：通过 `hasEbook` 布尔值判断
5. **简·爱人生阶段**：根据 `janeAge` 年龄三分支判断
6. **block wx:if**：`hasEbook` 控制电子书信息块
7. **wx:if vs hidden 对比**：展示两者区别

### 模板（template）使用位置
1. **bookItem 模板**：展示书籍章节信息
2. **characterInfo 模板**：展示人物基本信息
3. **maleCharacter 模板**：男性角色模板（蓝色背景）
4. **femaleCharacter 模板**：女性角色模板（粉色背景）
5. **动态模板切换**：根据 `char.gender` 动态选择模板

### 交互效果
- **按钮点击阴影收缩**：`hover-class` + CSS `transform: scale(0.96)` + `transition`
- 应用于"展开全部/收起"按钮，提升用户体验
