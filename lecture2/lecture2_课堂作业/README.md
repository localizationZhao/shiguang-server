# lecture2_课堂作业

## 作业内容

JSON数据格式调研

## 核心要点

### JSON简介

- 轻量级数据交换格式，独立于编程语言
- 基于ECMAScript子集，采用文本格式存储和表示数据
- 已成为IETF/ECMA国际标准

### 语法结构

- **对象**：使用`{}`包围，键值对集合
- **数组**：使用`[]`包围，值的有序列表

### 支持的数据类型

| 类型 | 说明 |
|------|------|
| String | 字符串 |
| Number | 数字 |
| Boolean | 布尔值 |
| Null | 空值 |
| Object | 嵌套对象 |
| Array | 数组 |

### 使用场景

- 前后端数据交互（Web API）
- 配置文件（settings.json、package.json）
- 数据存储（NoSQL数据库）
- 跨系统数据同步

### 优点

- 轻量级：比XML体积小30%以上
- 易于阅读和编写
- 跨语言兼容
- 与JavaScript原生兼容

### 局限性

- 不支持注释
- 不支持日期、函数等类型
- 需防范JSON注入攻击
