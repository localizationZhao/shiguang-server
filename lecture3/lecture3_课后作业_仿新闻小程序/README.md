# Lecture3_课后作业_仿新闻小程序

## 项目简介

将网络请求改造为本地数据操作的新闻小程序，支持频道管理、文件存储、下拉刷新等功能。

---

## 核心功能

| 功能 | 实现 |
|------|------|
| **本地数据** | `allData` 存储全部新闻，`filter()` 分类过滤 |
| **文件操作** | `wx.setStorageSync` / `wx.getStorageSync` 持久化频道配置 |
| **网络模拟** | `setTimeout` 模拟延迟 + `wx.showLoading` 加载提示 |
| **频道管理** | 添加/删除频道，数据实时同步到本地存储 |
| **交互反馈** | `wx.showToast`、下拉刷新、空状态提示 |
| **页面跳转** | `wx.navigateTo` + URL 参数传递新闻对象 |

---

## 技术要点

- 数据更新严格使用 `setData()`
- 废弃 `<loading>` 组件，改用 `wx:if` 条件渲染
- `scroll-view` 设置 `enable-flex` 使 flex 布局生效
- WXML 表达式不支持箭头函数，过滤逻辑在 JS 层完成
