# lecture3

## 目录结构

```
lecture3/
├── lecture3_课后作业_仿新闻小程序/    # 课后作业：仿新闻小程序
├── lecture3_课堂作业_视图容器/        # 课堂作业：视图容器组件
└── lecture3_课堂练习一和二/           # 课堂练习：数据绑定与事件
```

## 子文件夹概述

### lecture3_课后作业_仿新闻小程序

仿新闻小程序。包含首页、关注、视频、我的四个Tab页面，支持新闻列表展示、新闻详情页、用户中心等功能，使用scroll-view、swiper等视图容器组件。

### lecture3_课堂作业_视图容器

视图容器组件练习。实现swiper轮播图、scroll-view滚动容器等组件的使用，包括轮播配置、滚动视图属性设置等知识点。

### lecture3_课堂练习一和二

数据绑定与事件练习。包含两个课堂练习：数据绑定与运算（Mustache语法、变量绑定）、事件绑定与setData（事件处理、数据更新）。
