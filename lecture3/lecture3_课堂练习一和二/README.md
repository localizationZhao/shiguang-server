# lecture3_课堂练习一和二


## 子文件夹概述

### lecture3_课堂练习一_数据绑定与运算

数据绑定与运算练习。使用 Mustache 语法实现页面数据绑定，包括变量绑定、运算表达式、条件渲染等基础知识点。

### lecture3_课堂练习二_事件绑定与setData

事件绑定与setData练习。实现组件事件绑定（bindtap等）、setData数据更新、动态修改页面数据等核心功能。
