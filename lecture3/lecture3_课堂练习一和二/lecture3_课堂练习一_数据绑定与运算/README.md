# Lecture 3 课堂练习一：数据绑定与运算

## 项目简介

微信小程序基本数据绑定与页面数据运算练习，覆盖 Mustache 语法、各类数据类型绑定、算术运算、列表渲染、条件渲染等核心知识点。

---

## 功能模块

| 模块 | 内容 | 技术点 |
|------|------|--------|
| **练习1** | 基本数据绑定 | 10 种 JS 数据类型在 WXML 中的绑定 |
| **练习2** | 页面数据运算 | 算术、三元表达式、条件比较、逻辑运算 |
| **补充1** | 乘法表倒序 | `wx:for` 嵌套渲染、`onLoad` 生成数据 |
| **补充2** | 随身物品列表 | 自定义 `wx:for-index` / `wx:for-item` |
| **补充3** | 设备对象渲染 | 对象属性路径访问、规格列表展示 |
| **补充4** | 条件渲染图片 | `wx:if/wx:elif/wx:else` 根据设备类型显示不同图片 |

### 练习1：基本数据绑定（10 种类型）

- String、Number（整数/浮点数）、Boolean（true/false）
- null、undefined
- 一维数组（`wx:for` + `wx:key="*this"`）
- 嵌套对象（属性路径访问）
- 动态属性绑定（`id="item-{{age}}"`）

### 练习2：页面数据运算（4 类运算）

- **算术运算**：加、减、乘、除、取余
- **三元表达式**：`hidden` 属性控制、文本动态渲染
- **条件比较**：大于、小于、等于、不等于（`wx:if` 控制）
- **逻辑运算**：与、或、非（`wx:if` 多条件组合）

### 补充功能

- **乘法表倒序**：9×1~9×9 到 1×1，`onLoad` 时 JS 生成数据
- **随身物品列表**：自定义下标 `idx` 和列表项 `item`，展示图标+名称
- **设备对象渲染**：品牌、型号、CPU、内存等配置信息卡片
- **条件渲染图片**：根据 `deviceType` 值（apple/android/windows/other）显示对应图片

---

## 技术要点

- Mustache 语法 `{{}}` 支持简单表达式运算
- `<` 符号在 WXML 文本中需转义为 `&lt;`
- `wx:key="*this"` 用于纯数字/字符串数组
- 嵌套 `wx:for` 需用 `wx:for-item` 区分内外层变量名
- `hidden` 与 `wx:if` 区别：`hidden` 仅控制 CSS 显隐，`wx:if` 控制 DOM 存在与否

## 图片资源

`/images` 文件夹需放入以下图片（可选）：
- `apple.png` - 苹果 logo
- `android.png` - 安卓机器人
- `windows.png` - Windows 窗户

没有图片时条件渲染模块会显示"未知设备"占位，不影响其他功能。
