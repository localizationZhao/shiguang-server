# 图片资源说明

请将以下图片放入此文件夹：

| 文件名 | 内容 | 用途 |
|--------|------|------|
| `apple.png` | 苹果 logo | 苹果设备条件渲染展示 |
| `android.png` | 安卓机器人 | 安卓设备条件渲染展示 |
| `windows.png` | Windows 窗户 | Windows 设备条件渲染展示 |

**获取方式**：
1. 从网上搜索对应图片
2. 保存到本文件夹
3. 确保图片格式为 PNG/JPG，建议尺寸 200×200px

**临时方案**：如果没有图片，页面会显示"未知设备"占位区域，不影响其他功能。
