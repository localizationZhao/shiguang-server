# Lecture3_课堂练习二: 事件绑定与 setData

## 项目简介

微信小程序事件绑定与 setData 数据更新练习，覆盖基础事件、数据更新、事件传参、冒泡事件等核心知识点。

---

## 功能模块

| 模块 | 功能 | 技术点 |
|------|------|--------|
| **模块1** | 基础事件绑定 | `bindtap` 点击 view/button |
| **模块2** | setData 数据更新 | 数字自增、文本切换、数组增删、对象属性更新 |
| **模块3** | 事件传参 | `data-*` 自定义属性传参 |
| **模块4** | 条件与列表联动 | `wx:if` 控制列表显示/隐藏 + 列表项选中切换 |
| **模块5** | 冒泡事件 | `bindtap` 冒泡 vs `catchtap` 阻止冒泡 |

---

## 技术要点

- `bindtap` 事件会向父节点冒泡，`catchtap` 阻止冒泡
- 事件传参通过 `data-*` 自定义属性，在 `e.currentTarget.dataset` 中获取
- 数据更新必须使用 `setData()`，直接修改 `this.data` 不会触发视图更新
- 对象属性更新可使用路径语法：`this.setData({ 'user.name': '新值' })`
